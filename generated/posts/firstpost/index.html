Hello!

Quill is a markdown-powered personal blog built around a simple idea: Write in Markdown. Build. Publish. Posts are written as .md files, processed by the build system, and converted into static HTML pages.

Posting your blog

make sure to use the following as frontmatter for any of your blog posts

---
title: "My New Post"
date: "2026-07-17"
description: "A short description of the post."
tags:
  - programming
  - javascript
---
